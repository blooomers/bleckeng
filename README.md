# Maison des Images

An elegant landing page for a Photography and Film studio centered around the art world in Paris.

## Design

- **Aesthetic**: Black and white, elegant, design-oriented
- **Typography**: 
  - **Oswald** for headings and navigation (sans-serif)
  - **Libre Baskerville** for body text and serif elements
- **Style**: Minimalist, art world aesthetic fitting Paris contemporary scene

## Tech Stack

- **Next.js 16** with App Router
- **TypeScript**
- **TailwindCSS v4**
- **shadcn/ui** components
- **React**

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Fixed navigation with smooth scroll
- Hero section with elegant typography
- Services showcase
- About section
- Contact section with CTA
- Fully responsive design
- Elegant hover states and transitions

## Structure

- `/app/page.tsx` - Main landing page
- `/app/layout.tsx` - Root layout with font configuration
- `/app/globals.css` - Global styles and CSS variables
- `/components/ui/` - shadcn/ui components (when needed)
